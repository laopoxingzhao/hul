# Hul 语言语法文档

本文档描述 `hul` 语言的具体语法特性、语法结构和示例代码。

## 1. 基本语法元素

### 注释

- 单行注释：`// ...`
- 注释会一直跳过到行尾。

```hu
// 这是注释
let x = 1; // 变量声明
```

### 标识符

- 标识符由字母或下划线开头，后续字符可以是字母、数字或下划线。
- 用于变量名、函数名和参数名。

```hu
let _value = 42;
let x1 = 10;
```

### 字面量

- 数字：支持整数和浮点数，例如 `42`、`3.14`
- 字符串：双引号包裹，例如 `"hello"`
- 布尔值：`true`、`false`
- 空值：`nil`

```hu
let a = 123;
let s = "hul";
let ok = true;
let empty = nil;
```

## 2. 关键字

`hul` 支持以下保留关键字：

- `let`
- `if`
- `else`
- `while`
- `print`
- `fn`
- `return`
- `and`
- `or`
- `not`

这些关键字不能作为普通标识符使用。

## 3. 语句

### 3.1 变量声明

```hu
let x = 10;
let y = x + 5;
let z;
```

- `let` 后跟变量名。
- 可选初始化表达式；如果未提供初始化，则默认值为 `nil`。
- 语句必须以 `;` 结尾。

### 3.2 赋值语句

```hu
x = 20;
a = b = 3;
```

- 赋值目标必须是变量。
- 支持链式赋值。

### 3.3 打印语句

```hu
print x;
print "hello";
print x + 1;
```

### 3.4 条件语句

```hu
if (x > 0) {
    print "positive";
} else {
    print "non-positive";
}
```

- `if` 后必须跟 `()` 包裹的条件表达式。
- `then` 分支必须是代码块 `{ ... }` 或单条语句。
- 可选的 `else` 分支也可以是代码块或单条语句。
- `else if` 语义通过嵌套 `if` 实现：

```hu
if (x > 0) {
    print "positive";
} else if (x < 0) {
    print "negative";
} else {
    print "zero";
}
```

### 3.5 循环语句

```hu
while (i < 10) {
    print i;
    i = i + 1;
}
```

- `while` 后必须跟 `()` 包裹的循环条件。
- 循环体可以是代码块或单条语句。

### 3.6 代码块

```hu
{
    let a = 1;
    print a;
}
```

- 代码块使用 `{}` 包裹多条语句。
- 代码块会创建新的作用域。

### 3.7 函数声明与返回

```hu
fn add(a, b) {
    return a + b;
}

let result = add(2, 3);
print result;
```

- 函数声明使用 `fn` 关键字。
- 参数列表用 `()` 包裹，参数间用 `,` 分隔。
- 函数体必须为代码块。
- `return` 用于从函数返回值，必须以 `;` 结尾。

### 3.8 表达式语句

```hu
x + 1;
```

- 任何表达式后面跟 `;` 都可视为表达式语句。
- 赋值表达式会被解析为 `Assign` 语句。

## 4. 表达式

### 4.1 主表达式

- 字面量：`42`, `3.14`, `"text"`, `true`, `false`, `nil`
- 变量：`x`, `name`
- 分组：`(expr)`
- 函数调用：`f(a, b)`

### 4.2 运算符

#### 4.2.1 一元运算符

- `-x`：数值取负
- `not x`：逻辑非

#### 4.2.2 算术运算符

- `+`：加法
- `-`：减法
- `*`：乘法
- `/`：除法
- `%`：取模

#### 4.2.3 比较运算符

- `==`：等于
- `!=`：不等于
- `<`：小于
- `<=`：小于等于
- `>`：大于
- `>=`：大于等于

#### 4.2.4 逻辑运算符

- `and`：逻辑与
- `or`：逻辑或

#### 4.2.5 运算符优先级（从低到高）

1. 赋值 `=`（右结合）
2. 逻辑或 `or`
3. 逻辑与 `and`
4. 相等 `==`, `!=`
5. 比较 `<`, `<=`, `>`, `>=`
6. 加减 `+`, `-`
7. 乘除取模 `*`, `/`, `%`
8. 一元 `-`, `not`
9. 函数调用、分组、变量、字面量

### 4.3 函数调用

```hu
fn greet(name) {
    print name;
}

greet("hul");
```

- 函数调用使用 `()` 和逗号分隔参数。
- 支持嵌套调用和表达式作为参数。

## 5. 语法示例

### 5.1 计算示例

```hu
let x = 1 + 2 * 3;
print x; // 输出 7
```

### 5.2 条件与循环示例

```hu
let sum = 0;
let i = 1;
while (i <= 5) {
    sum = sum + i;
    i = i + 1;
}
print sum; // 输出 15
```

### 5.3 函数示例

```hu
fn factorial(n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}
print factorial(5); // 输出 120
```

- `==`  `!=`  `<`  `<=`  `>`  `>=`
- `=`  `;`  `,`  `(`  `)`  `{`  `}`

- `let` `if` `else` `while` `print` `fn` `return` `and` `or` `not`

### 文字

- `Number`：数字字面量
- `String`：字符串字面量
- `True`：`true`
- `False`：`false`
- `Nil`：`nil`

## 7. 备注

- 所有语句必须以分号 `;` 结尾（除了由代码块或条件/循环控制结构直接包裹的语句）。
- `else if` 是通过嵌套 `if` 实现的。`
- 当前语法未实现类、数组、对象或模块等高级特性。
